**Класс** - Product

**Защищённые атрибуты экземпляра** - имя, цена, состояние и кол-во на складе.
## Управление доступом через @property

Состояние экземпляра контролируем через @property:
- **Геттер (@property)** - контролируем чтение
- **Сеттер (@price.setter)** - контролируем изменение с проверками

Геттеры я сделала для всех атрибутов, чтобы можно было получить их значения. Сеттер сделала только для цены, потому что цена может меняться.

## Атрибуты класса

**Атрибуты класса** - скидка и ндс

## Валидация (файл validate.py)

В файле Validate я вынесла все проверки через @classmetod, такие как:
- **Проверка типа данных** 
- **Проверка диапазона** 
- **Проверка допустимых значений** (статус должен быть из списка)

## Магические методы

**Магические методы:**
- **`__str__`** - красивый вывод для пользователя

- **`__repr__`** - вывод для разработчика

- **`__eq__`** - сравнение товаров

## Бизнес-методы

**Бизнес методы:**
- **общая стоимость товара** (`total_value()`)
- **проверка возможности продажи** (`can_be_sold()`)
- **покупка** (`purchase()`)
- **пополнение запасов** (`restock()`)
- **снятие с производства** (`discontinue()`)
- **возврат в продажу** (`activate()`)

## Логическое состояние

**Логическое состояние:**
