### Реализованный класс коллекции: ProductCatalog

Коллекция для хранения и управления объектами Product из ЛР-1. Реализует контейнер для группы товаров с возможностью добавления, удаления, поиска, сортировки и фильтрации.

### Краткое описание класса

ProductCatalog — это контейнерный класс для управления коллекцией товаров. Внутри хранит список объектов Product.

#### Основные возможности:

- **Хранение** — список объектов Product
- **Управление** — добавление, удаление, доступ по индексу
- **Поиск** — по названию, ценовому диапазону, статусу
- **Сортировка** — по названию, цене, количеству
- **Фильтрация** — получение подколлекций (в наличии, снятые, дорогие, дешевые)
- **Итерация** — поддержка цикла for
- **Защита** — проверка типа, запрет дубликатов

### Методы коллекции

#### Базовые операции:
- `add(item)` — добавить товар
- `remove(item)` — удалить товар
- `remove_at(index)` — удалить по индексу
- `get_all()` — получить список всех товаров

#### Поиск:
- `find_by_name(name)` — поиск по названию
- `find_by_price_range(min, max)` — поиск по диапазону цен
- `find_by_status(status)` — поиск по статусу

#### Магические методы:
- `__len__` — получение количества
- `__iter__` — итерация
- `__getitem__` — доступ по индексу
- `__contains__` — проверка наличия

#### Сортировка:
- `sort_by_name()` — сортировка по названию
- `sort_by_price()` — сортировка по цене
- `sort_by_amount()` — сортировка по количеству
- `sort(key)` — универсальная сортировка

#### Фильтрация (возвращают новую коллекцию):
- `get_available()` — товары в наличии
- `get_discontinued()` — снятые товары
- `get_expensive(threshold)` — дорогие товары
- `get_cheap(threshold)` — дешевые товары
- `get_by_status(status)` — товары по статусу

#### Дополнительные:
- `total_value()` — общая стоимость всех товаров
- `clear()` — очистка коллекции
- `is_empty()` — проверка на пустоту
